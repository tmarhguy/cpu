# Miscellaneous Files

This directory contains files that don't fit into the main project structure but are still part of the repository.

## Contents

- **firmware/** - Arduino firmware code (controller-input, controller-display)
- **hardware/** - Hardware design notes and documentation
- **KiCad alias files** - macOS alias files for KiCad shortcuts
- **ALU Design 3.pdf** - Design document PDF

These files are preserved for reference but may be organized differently in future updates.
