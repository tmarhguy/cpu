## Naming Convention

</div>

<table>
<tr>
<td width="50%">

### Photo Files

**Format Examples**
- `build_photo_01.jpg`
- `debug_solder_bridge.jpg`
- `final_demo_setup.jpg`
- `component_placement_01.jpg`
- `circuit_detail_alu_core.jpg`

**Naming Pattern**
- Descriptive, lowercase
- Underscore separators
- Sequential numbering
- Clear subject identification

</td>
<td width="50%">

### Video Files

**Format Examples**
- `demo_add_operation.mp4`
- `test_vector_run.mp4`
- `system_overview.mp4`
- `subtraction_demo.mp4`
- `control_unit_test.mp4`

**Naming Pattern**
- Operation or test name
- Clear purpose indication
- Consistent format
- Chronological when applicable

</td>
</tr>
</table>

</div>

<table>
<tr>

- Good framing
- Appropriate length

</td>

- Provide context

<div align="center">

## Usage

</div>

Media files in this directory are referenced throughout the project documentation:

- **Hardware documentation**: Build process photos
- **Testing documentation**: Test execution videos
- **Architecture documentation**: System overview videos
- **Debugging logs**: Troubleshooting photos and captures

Ensure all media files are properly named and organized for easy reference.
